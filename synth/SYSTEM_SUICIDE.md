## 1. I SHOW YOU LIGHT — Оптический тон-колесо осциллятор

### Суть
Вращающийся диск с вырезами/паттернами модулирует луч света между LED и фотодиодом — физическое колебание, превращённое в звук через свет. Второй режим — контактный: пьезо-стилус снимает текстуру поверхности, как игла граммофона.

### Физический элемент
Алюминиевый или акриловый диск (∅60–80мм) с лазерной резкой паттернов на нескольких дорожках. Вращается BLDC-мотором (бесщёточный для stability) с оптическим энкодером на оси для PLL-lock скорости. LED с одной стороны, фотодиод — с другой. Паттерн прерывает свет → фотодиод генерирует аудиосигнал.

Принцип тот же, что в органах Hammond (тон-колесо), но:
- Один диск вместо 91 колеса.
- CV-управление скоростью мотора = плавный pitch с V/Oct tracking через expo converter + PLL.
- Сменные диски = разные тембры/формы волны.
- Намеренно lo-fi: артефакты мотора, биение диска, оптический шум — часть характера.

### Сигнальная цепь

```
CV PITCH (V/Oct) ──→ Expo converter ──→ Motor Driver ──→ BLDC Motor ──→ Диск (вращение)
                                              ↑                            │
                                     Optical encoder ──────────────────────┤
                                     (PLL-lock feedback)                   │
                                                                           │
                              LED ──→ [диск] ──→ Фотодиод (transimpedance)
                                                    │
                                              Waveshaping (компаратор/мягкий клип)
                                                    │
                                              Output Buffer → LIGHT OUT
                                     
                              Пьезо-стилус ──→ JFET preamp ──→ DUST OUT
                                 (контакт
                                  с диском)
                                                    │
                                     LIGHT ↔ DUST crossfade pot ──→ MAIN OUT
```

### Ключевые контролы

| Контрол | Функция | CV |
|---------|---------|-----|
| **PITCH** | Скорость мотора. Грубая настройка + fine. V/Oct tracking через PLL | ✓ (V/Oct) |
| **TRACK** | Выбор дорожки на диске (разные паттерны = разные формы волны) | — |
| **LIGHT / DUST** | Баланс между оптическим и контактным выходами | — |
| **BRIGHT** | Яркость LED (влияет на клиппинг фотодиода = тембр) | ✓ |
| **SYNC IN** | Жёсткая синхронизация через отдельную метку на диске | ✓ (gate) |

### Тембральный характер
Органный, слегка гудящий, с механическим дыханием. На низких скоростях — ритмичное щёлканье, переходящее в тон. На высоких — жирный, гармонически насыщенный звук с лёгкой нестабильностью pitch (вибрато от инерции мотора, даже при PLL остаётся <0.2%). Контактный режим добавляет скрип, треск, текстуру поверхности.

Не чистый VCO. Живой, тёплый, нестабильный. Ближе к шарманке, чем к Moog.

### Сменные диски (standardized формат ∅70мм × 1.5мм акрил или алюминий)
- **Sine** — плавные синусоидальные прорези (interferometric pattern).
- **Square** — равномерные щели (прямоугольник, скважность 50%).
- **Harmonics** — несколько дорожек с кратными частотами (аддитивный синтез).
- **Noise** — хаотичный паттерн (шумогенератор).
- **Custom** — пользовательские диски по SVG (service offered).

### Связь с Last Night
Органный тон I Show You Light → через фильтр и дисторшн → в Last Night создаёт звук утробного рокота, гула заброшенного собора. Контактный режим (DUST) + деревянная пластина Last Night = звук сломанного граммофона в пустой комнате.

### Сложность: Средняя-высокая
BLDC мотор + encoder + PLL — стандарт robotики, но требует калибровки. Оптика — простая. Самое сложное — изготовление точных дисков (лазерная резка по производственной схеме) и экспоненциальный конвертер CV→скорость мотора для V/Oct tracking.

**R&D**: 6–9 месяцев (после ship Phase 1 Last Night).

### Аналоги
Электромеханические осцилляторы существуют (Folktek Mescaline, некоторые DIY проекты), но именно оптическое тон-колесо в Eurorack со сменными дисками и двойным считыванием (свет + контакт) — не существует.

## 2. BODY BLOOD AND SALT — Касание тела

> *Phase 5, опциональный модуль*. Оригинальная концепция (аудио через биологические жидкости) требует medical-grade изоляции и certification — отложена. Текущая версия — упрощённая, только tactile, без жидкостей. Safety при этом уже не критична.

### Суть в одном предложении
Модуль, в котором прикосновение кожи становится частью аудиотракта — сопротивление кожи, влажность пальцев, площадь контакта модулируют параметры сигнала в реальном времени.

### Физический элемент
**TOUCH (пальцы):** 4 пары позолоченных контактных площадок на панели. Касаешься двумя пальцами — замыкаешь цепь через кожу. Сопротивление кожи (1 кОм – 1 МОм в зависимости от влажности, давления, площади) включено в аудиоцепь через current-limited buffer (защита по медстандарту <50 мкА).

### Функции Touch Pads

- **PAD 1 — GATE:** палец = gate on, убрал = gate off. Сопротивление кожи модулирует velocity.
- **PAD 2 — FILTER:** сопротивление кожи = cutoff RC-фильтра. Сухой палец = тёмный, мокрый = яркий.
- **PAD 3 — BEND:** CV-выход, пропорциональный проводимости контакта. Подключай куда хочешь.
- **PAD 4 — CHAOS:** соединена с feedback-петлёй. Прикосновение = самовозбуждение, степень определяется влажностью/давлением.

Можно касаться нескольких пар одновременно. Два человека: один держит один контакт, второй — другой, берутся за руки — замкнули цепь через два тела (через медленные DC currents <50 мкА, безопасно).

### Сигнальная цепь

```
AUDIO IN ──→ Input buffer (±1V attenuator — safety)
                │
                ├──→ PAD 1 (GATE) ──→ skin R → comparator → Gate/Velocity out
                │
                ├──→ PAD 2 (FILTER) ──→ skin R in RC → filtered audio out
                │
                ├──→ PAD 3 (BEND) ──→ skin R → CV out (±5V range)
                │
                └──→ PAD 4 (CHAOS) ──→ skin R → feedback amp → self-oscillation
                                                                  │
                                                    Recovery amp  │
                                                                  │
                                                            GAIN recovery
                                                                  │
                                                            DRY/WET ←── DRY
                                                                  │
                                                             OUT (audio)
```

### Ключевые контролы

| Контрол | Тип | Функция |
|---------|-----|---------|
| **TOUCH PADS 1–4** | Контактные площадки | Gate, Filter, Bend, Chaos — пальцами |
| **GAIN** | Потенциометр | Усиление после touch tract |
| **DRY/WET** | Потенциометр | Баланс чистого и обработанного сигнала |
| **SENSITIVITY** | Потенциометр | Скейл skin-R range под индивидуальный кожный импеданс |
| **FEEDBACK** | Потенциометр | Уровень обратной связи (CHAOS pad усиливает) |
| **LIFE** | LED | Яркость = суммарная проводимость = "life" индикатор |

### Тембральный характер
Живой, нестабильный, тактильный. Сопротивление пальца варьирует 1кΩ–1МΩ — три порядка величины, огромный control range. Каждое прикосновение уникально. Самовозбуждение через CHAOS pad — physical theremin feel. Два игрока, соединённые руками — одно общее вибрато-дыхание.

### Связь с системой
Стоит вторым в цепи — после рождения звука (I Show You Light). Свет родил звук → тело приняло, пропустило через кожу. Звук становится **human-touched**. Дальше — через кости (All Bones Dust) → через стекло (Be Careful) → в пространство (Last Day / Last Night).

### Оригинальная концепция (WELL с жидкостями) — Phase 5 roadmap
Ванночка с сменными электродами и жидкостями (слюна, пот, кровь, масло, мёд) с гальванической изоляцией по медстандарту — оригинальная идея **может вернуться в Phase 5** как premium SKU с полной safety certification. Но для v1 серии — только touch-pads.

### Сложность: Средняя
Touch-pads — стандартные схемы (guitar fuzz с palm contact есть как precedent). Безопасность: current limiter на каждом pad, isolation от power rails, FET multiplexing для избежания crosstalk между pads.

**R&D**: 4–6 месяцев.

### Аналоги
SOMA Laboratory (Srapa, Ether) используют тело как контроллер, но туда audio не проходит. Folktek — тактильные инструменты. Buchla Kinesthetic Input Port (разъём для skin-contact) — близко. Но интеграция skin-R напрямую в аудио-фильтр + feedback — уникально.

## 3. ALL BONES DUST — Трансформаторный сатуратор + раттл

### Суть
Двойная нелинейность: аудиотрансформатор, загнанный в насыщение (тёплые чётные гармоники), плюс металлическая/костяная пластина, вибрирующая об ограничитель (обертона ситара, jawari-buzz).

### Физический элемент
**Элемент 1 — Трансформатор:** Аудиотрансформатор (Lundahl LL1538 premium, или Triad TY-141P budget, или дешёвый 600:600 телефонный для lo-fi SKU). При перегрузке сердечник насыщается, формируя мягкий клиппинг с преобладанием чётных гармоник. Характер зависит от типа — сменные модули в слоте.

**Элемент 2 — Rattle (раттл):** Тонкая пластина (металл, кость, рог, дерево) закреплена над пьезо-диском с регулируемым зазором. Сигнал возбуждает пьезо, пластина вибрирует и бьётся об ограничитель при превышении амплитуды. Зазор регулируется винтом-каподастром с CV-моторизованным приводом (soft servo). Это принцип jawari из ситара / tanpura — создаёт каскад обертонов, зависящих от формы и материала пластины.

### Сигнальная цепь

```
AUDIO IN ──→ Input Gain (DRIVE) ──→ ┬──→ Трансформатор (saturation) ──→ ┐
                                     │                                    │
                                     └──→ Exciter ──→ [пластина-раттл] ──→ Piezo ──→ ┐
                                                                                      │
                                                                                      │
                                           BLEND ←── Crossfader ←────────────────────┘
                                              │
                                         Tone control (tilt EQ)
                                              │
                                         Output Buffer → OUT
```

### Ключевые контролы

| Контрол | Функция | CV |
|---------|---------|-----|
| **DRIVE** | Уровень входного сигнала. Определяет глубину насыщения трансформатора и амплитуду раттла | ✓ |
| **BUZZ** | Зазор между пластиной и ограничителем. Ближе = больше раттла, частотно-зависимая грязь | ✓ |
| **BLEND** | Баланс трансформатор / раттл | ✓ |
| **TONE** | Пост-дисторшн тилт EQ (bright ↔ dark) | — |

### Сменные раттл-элементы
Пластина на защёлке — быстрая замена:
- **Латунь** — яркий, звонкий buzz с длинным sustain.
- **Кость / рог** — сухой, щёлкающий, перкуссивный.
- **Тонкая сталь 0.3мм** — агрессивный, пилящий.
- **Дерево (бук, клён)** — мягкий, тёплый rattle.

### Тембральный характер
Трансформаторный путь: тёплый, ламповый сат, как раскачанный магнитофон. Приятная компрессия на басах.
Раттл-путь: ситарный buzz, расщепление нот на обертоновые каскады, живая грязь.
Вместе: звук мясорубки, перемалывающей колокола. Или тихое потрескивание старых костей.

### Связь с Last Night
All Bones Dust → Last Night = индустриальный нойз-эмбиент. Раттл-обертона, пропущенные через деревянную или каменную пластину Last Night, превращаются в гудящий, зернистый шлейф. Трансформаторная сатурация перед ревербом добавляет плотности и "аналоговой" теплоты хвосту.

### Сложность: Низкая — **первый модуль в ship Phase 1**
Трансформатор — купить готовый. Раттл-механизм — пьезо-диск, полоска металла, регулировочный винт с servo за вечер. Электроника минимальна: буферы, микшер, тон-контроль.

**R&D**: 2–3 месяца (быстрее всех остальных). Ship одновременно с Last Night в Phase 1.

### Аналоги
Трансформаторные сатураторы существуют (Strymon Deco, Radial, в Eurorack — AI Synthesis). Механический раттл/jawari в педалях не реализован вообще (есть в ситарах, но не как эффект-процессор). Комбинация двух физических нелинейностей с бленд-контролем — уникальна.

## 4. BE CAREFUL — Стеклянный резонансный фильтр

> *"Be careful" — стекло хрупкое.*

### Суть
Аналог Last Night в формате фильтра: вместо сменных пластин из разных материалов — **стеклянные резонаторы с высоким Q**. Exciter возбуждает пластину, пьезо снимает резонанс. Управляемая обратная связь превращает резонатор в **self-oscillating band-pass фильтр** с выраженным стеклянным характером.

### Физический элемент
Стеклянная пластина (Pyrex или borosilicate, 100×40×3мм) закреплена в раме через резиновые уплотнители (Q-preserving mount). Exciter (Dayton DAEX13CT-8, 13мм, light-touch для хрупкого стекла) приклеен/прижат к face A. Два пьезо-диска 20мм на face B на разных позициях.

Пластина сменяется картриджем, как в Last Night — но **только стеклянные варианты**:
- **Pyrex 3мм** (стандарт): чистый резонанс, высокий Q, ~1кГц fundamental.
- **Soda-lime glass**: тише, грязнее, исторический аромат.
- **Tempered glass**: максимальный Q, длинный sustain, но риск explosive breakage при resonance peak.
- **Crystal (свинцовое стекло)**: звенящий, "bell-like", премиум.

### Сигнальная цепь

```
AUDIO IN ──→ Input Gain ──→ Exciter driver ──→ Стеклянная пластина
                                                     │
                                              Piezo A (bright)
                                              Piezo B (warm)
                                                     │
                                              Position crossfade
                                                     │
                                              Feedback loop ──→ RESONANCE knob (Q)
                                                     │
                                              Cutoff: скорость motor-clamp на пластине
                                                     │                   (shifts effective length)
                                                     │
                                              Output Buffer → OUT
```

### Ключевые контролы

| Контрол | Функция | CV |
|---------|---------|-----|
| **CUTOFF** | Положение motor-clamp на пластине (меняет эффективную длину → сдвигает резонансную частоту в диапазоне ~500Гц–3кГц) | ✓ (V/Oct в узком range) |
| **RESONANCE / Q** | Feedback gain → self-oscillation на max | ✓ |
| **DRIVE** | Уровень возбуждения пластины | — |
| **POSITION** | Кроссфейд между piezo A/B (bright ↔ warm) | — |
| **TRACK SELECT** | Выбор картриджа (если установлен auto-detect) | — |

### Тембральный характер
Узкополосный резонанс с ярко выраженным "колокольным" характером стекла. При высоком RESONANCE — self-oscillating sine с лёгкой гармонической структурой (стекло имеет не только fundamental, но и overtones 2f, 3f с amplitude ~-20dB). На low resonance — мягкий band-pass с сохранением dry signal.

Звучит **как резонансный фильтр с крошечной комнатой реверберации внутри** — каждая нота оставляет короткий glass-tail (~100мс).

### Связь с системой
Be Careful стоит **после** All Bones Dust в цепи. Грязь трансформатора и костяной раттл → пропускаются через стеклянный фильтр → получаем **crystalline distortion** с узкополосным resonance. Self-oscillating Be Careful + Last Night freeze → drone пустого собора.

### Сложность: Средняя
Reuses Last Night cartridge infrastructure (80% overlap в R&D). Отличия:
- Stepper motor + lead screw для motorized cutoff (новая механика).
- Feedback loop с уверенным limiter (глёсок стекла при oscillation может скакать ±10dB).
- Хрупкость transport: каждый картридж в foam-padded case.

**R&D**: 6–9 месяцев (Phase 3, после Last Night infrastructure proven).

### Аналоги
- Folktek Nano/Micro используют piezo + objects, но не как фильтр.
- Spring reverb tanks со swappable springs — concept близок, но filter character другой.
- **Стеклянный резонансный фильтр в Eurorack — не существует**. Unique.

### Предостережение
Стекло может треснуть при self-oscillation пиках. Cartridge замена — разбитая пластина → покупка новой ($15–30). Warranty не покрывает breakage from over-drive. "Be careful" — не метафора.

## 5. FUCK ABANDONED SLEEP — Маятниковый LFO

### Суть
Физический маятник качается внутри модуля — оптический датчик считывает его положение и генерирует управляющее напряжение. Период колебания определяется длиной маятника. При отсутствии подпитки — естественное затухание. При хаос-режиме (двойной маятник) — детерминированный chaos.

### Физический элемент
**Маятник** — латунный грузик (20г) на карбоновом стержне 2мм, подвешенный на миниатюрном шариковом подшипнике. Длина подвеса регулируется сдвигом грузика по стержню: 2–10 см → период 0.3–0.6с → частота 1.5–3.5 Гц (идеальный LFO-диапазон).

**Считывание** — IR LED + позиционно-чувствительный детектор (PSD, Hamamatsu S3932). Положение маятника → линейное напряжение ±5В. Естественная форма — синус с постепенным затуханием от трения.

**Поддержание колебаний** — электромагнит у основания, управляемый PWM от MCU. Feedback от PSD определяет момент push-а (как в электронных часах). SUSTAIN knob регулирует gain feedback: 0 = decay, 1 = perpetual motion.

**Хаос-режим** — второй маятник крепится к первому через шарнир с magnetic release. При CHAOS = OFF — магнит удерживает второй маятник rigid. При CHAOS = ON — магнит отпускает, система становится double pendulum (классический хаос).

### Сигнальная цепь

```
RATE (длина маятника) ──→ Маятник (физическое качание)
                              │
SUSTAIN ──→ Электромагнит ────┘ (подталкивает в нужный момент)
                              │
                    IR LED ──→ [маятник] ──→ PSD
                                              │
                                        Analog conditioning
                                           │         │
                                      RAW OUT    Waveshaper
                                                (comparator,
                                                 rectifier)
                                                     │
                                                SHAPED OUT
                                                (tri/sq/gate)
```

### Ключевые контролы

| Контрол | Функция | CV |
|---------|---------|-----|
| **RATE** | Положение грузика на стержне = длина маятника = период | — (механический) |
| **SUSTAIN** | Сила электромагнита. Минимум = маятник затухает (одноразовый envelope). Максимум = бесконечное качание | ✓ |
| **CHAOS** | Переключатель ON/OFF отпускает второй маятник | ✓ (gate) |
| **SHAPE** | Обработка CV: raw sine → triangle → square → random gate (из хаотического режима) | — |
| **AMPLITUDE** | Масштаб выходного CV (аттенюатор) | — |
| **TRIGGER IN** | Ручной толчок маятнику (gate в электромагнит pulse) | ✓ (trigger) |

### Два выхода
- **WAVE** — плавное CV (синус/треугольник, ±5В).
- **GATE** — компараторный gate/trigger (настраиваемый порог).

### Тембральный характер (как модулятор)
Модуляция с физической инерцией. Маятник не может мгновенно изменить скорость — нет резких переходов, только плавные ускорения и замедления. При затухании (low SUSTAIN) — естественный decrescendo, как замирающее дыхание. В хаос-режиме — непериодическое, но не случайное (детерминированный хаос, паттерны есть, но не повторяются).

Визуально: маятник виден через окошко в панели (закалённое стекло 2мм, anti-reflection coating). Перформер и зритель видят движение — модуляция становится зрелищной.

### Связь с системой
- → I Show You Light PITCH: slow pendulum = vibrato, chaos = random pitch drift.
- → Last Day HAZE: маятник управляет wow/flutter delay.
- → Last Night DECAY: затухающий маятник = реверб, который сначала длинный, потом всё короче, и наконец замирает.
- → Be Careful CUTOFF: маятниковый sweep по стеклянному фильтру.

### Сложность: Средняя
Механика маятника проста, но требует точной балансировки (ось шарикоподшипника, minimized friction). Электромагнитный привод — standard PWM. Оптика — IR LED + PSD ($15 sensor). Сложность: калибровка, минимизация vibrations от корпуса, mechanics двойного маятника с magnetic release.

**R&D**: 4–6 месяцев. Phase 3 (after Phase 1 ship).

### Аналоги
Маятниковые модуляторы существуют как арт-объекты и инсталляции, но в формате Eurorack с CV-выходом, электромагнитным приводом, chaos-режимом и visible window — **не реализовано**. Ближайшее: Landscape Stereo Field (тактильный controller) — не generator CV.

## 6. IS MY — Оптический кроссфейдер

### Суть
Плавный CV-управляемый кроссфейд между двумя аудиосигналами через пару vactrol'ов. Lo-fi "оптический" характер — не идеально линейный, не equal-power, с лёгкой оптической инерцией от LDR. Для live-performance — между двумя голосами, для перехода Last Day → Last Night, для slow morph в ambient patches.

### Физический элемент
Два **vactrol'а** (VTL5C3 или VTL5C4 matched pair) — LED внутри opaque housing светит на LDR (photoresistor). Инерция LDR ~10–50мс даёт optical lag, который **слышен как smooth-lag crossfade**.

CV 0В → выход только A. CV 5В → выход только B. CV 2.5В → ~equal mix (non-linear, slightly lagging from optical inertia).

### Сигнальная цепь

```
AUDIO A ──► Input A buffer ──► Vactrol A (LDR1) ──┐
                                                   ├──► Summing amp ──► OUTPUT
AUDIO B ──► Input B buffer ──► Vactrol B (LDR2) ──┘

  CV ──► LED driver ──┬──► LED in Vactrol A (inverted for A attenuate)
                      └──► LED in Vactrol B (non-inverted for B pass)

  Manual knob ──► mixed with CV at CV summing point
```

### Ключевые контролы

| Контрол | Функция | CV |
|---------|---------|-----|
| **POSITION** | Ручной baseline кроссфейда (0 = A, 10 = B, 5 = mix) | — |
| **CV amount** | Аттенювертер CV input | — |
| **CURVE** | Переключатель: linear / equal-power / reverse-log | — |
| **SYNC input** | Gate-triggered A/B switch (momentary) | ✓ |
| **LED indicator A** | Visual показывает уровень vactrol A | — |
| **LED indicator B** | Visual показывает уровень vactrol B | — |

### Тембральный характер
**Оптическая инерция** — это главная фишка. Быстрый CV-sweep (<30мс) получает lag, делает crossfade "springy". Медленный sweep — работает как стандартный VCA. При быстрых gate-переключениях (SYNC input) — audible "fade-in-fade-out" 30мс вместо hard switch.

Характер: "рассказ 1970-х аналогового мира" — не идеально чистый, но музыкально приятный. Добавляет немного mid-range EQ (LDR не flat).

### Связь с системой
- **Главная роль**: live crossfade между двумя голосами.
- **Перфоманс**: triggered SYNC = rhythmic A/B jumping в такт секвенсору.
- **Ambient**: slow CV → slow morph между двумя reverb tails.

### Почему это **не** clock divider и **не** octaver
В оригинальной концепции были три conflicting функции в одном 6HP модуле. После decision — одна функция: vactrol crossfade. Это решение освобождает 6HP для чистой ergonomики, а clock utilities и octaver — стандартные модули, легко найти у других производителей.

### Сложность: Низкая
5 компонентов + 2 vactrol + 2 pot + 2 jack input + 1 jack output + 1 CV input + 1 sync input. ~30 компонентов total. 1-layer PCB достаточна.

**R&D**: 2–3 месяца. Phase 2 (параллельно с Last Day).

### Аналоги
- Mutable Instruments Blinds / Veils — quad VCA с linear response, но без оптической инерции.
- Joranalogue CV Mix — similar, без vactrol character.
- Happy Nerding 3×VCA — classic op-amp VCA, no optical.
- **Vactrol-based crossfader в Eurorack 6HP** — есть в DIY, но не в commercial market. Unique character.
